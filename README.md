# ScriptRun - macOS 的 JavaScript 自動化工具

ScriptRun 是一款為 macOS 設計的自動化工具，使用 JavaScript 輕鬆創建各種自動化腳本。

## 核心特點

### 系統整合：提供預製的系統 API，輕鬆在 JavaScript 函式中操作系統功能
 - 檔案操作（輸入、寫入、執行指令）
- 圖片處理（輸入、獲取資訊、轉為為 Base64 輸入至 JavaScript 中執行）
- 資料庫操作（SQL 查詢、參數化查詢）

### 內建資料庫支援（SQLite）
- 獨立的 SQLite 資料庫管理器
- 視覺化資料表設計
- 資料導入導出
- 在腳本中直接使用 SQL

### 快速存取
- 選單列快速動作
- 全域快捷鍵支援
- 釘選常用腳本

### 社區分享
- 匯出／匯入腳本
- 分享代碼
